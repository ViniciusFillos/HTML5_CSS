Favicons

Favicons podem ser traduzidos como icones favoritos, são icones que ficam dispostos no topo da aba do navegador antes do título.

Como só podemos utilizar 1 favicon por página web normalmente deifinimos seu nome como favicon.io

Para encontrar favicons podemos utilizar dos seguintes sites:

  • ICON ARCHIVE

    • Para fazer pesquisas de favicons

  • FAVICON.CC

    • Para desenhar seu próprio favicon

  • FAVICON.IO

    • Para tranformar um texto, emoji ou arquivo png em favicon

A tag para utilizar um favicon é definida dentro da tag <head> </head>, logo antes do título acrecentamos a linha:

<link rel="shortcut icon" href="img/favicon.ico" type="image/x-icon">

Para facilitar o processo podemos apenas digitar link e uma lista irá aparecer, nela procuramos por link:favicon.
