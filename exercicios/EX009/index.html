Orders Lists

Para criar uma lista ordenada utilizamos as tags < ol > < /ol > para criar a lista e a tag < li > para definir os seus itens.

Unorder Lists

Para criar uma lista não ordenada utilizamos as tags < ul > < /ol > para criar a lista e a tag < ul > para definir os seus itens.

  • CSS

Parâmetros

Dentro das tags de definição de uma lista podemos utilizar os parametros type para definir o tipo de contagem ou o formado da bolinha em listas não ordenadas, nas listas ordenadas também podemos usar o parâmetro start definindo de qual número a lista começará a contar.
