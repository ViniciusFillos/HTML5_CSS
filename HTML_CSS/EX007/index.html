Principais Formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando as tag < b > < /b > (não semânticas).

Nessa frase tenho um termo em destaque usando a tag < strong > < /strong >(semânticas).

Itálico / Ênfase

Nesta frase temos um termo em itálico usando as tag < i > (não semânticas).

Nesta frase temos um termo em ênfase usando as tags < em > (semântica).

Texto marcado

Podemos também criar um texto marcado usando as tags < mark > < /mark >

E no outro parágrafo temos outro texto marcado.

Texto grande e pequeno

Estamos criando um texto grande usando as tags < big > < /big > e um texto pequeno usando as tags < small > < /small > neste parágrafo.

Texto Excluído

Podemos marcar um texto como deletado, para indicar que ele pode ser lido mais não considerado utilizando as tags < del > < /del >

Texto inserido

Podemos marcar um texto como inserido para dar ênfase e indicar que ele foi inserido depois utilizando as tags < ins > < /ins >

Existe também a tag < u > < /u > que sublinham porém sem valor semântico.

Texto sobrescrito

Para inserir coisas tipo x2 + x1 podemos utilizar as tags < sup > < /sup >

Texto subescrito

Para inserir coisas do tipo H2O podemos utilizar as tags < sub > < /sub >
