Usando links externos

Você pode acessar meu repositório público no GitHub.

Para definir um link usamos as tags < a > < /a >

Também usamos a propriedade target=_"blanck" ou _"self" para abrir em outra aba e a propriedade rel="next", "prev" ou "external" para dizer se é um próximo documento, anterior ou de outro site. Também há um atributo para definir o idioma do link, hreflang="pt-br"
